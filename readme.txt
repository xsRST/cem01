1.用户登录实现
  后台service(重点)
     1.参数校验
        用户名  密码 非空
     2.执行查询
        根据用户名查询用户记录
     3.记录存在校验  密码匹配
        记录非空
        密码加密比对
     4.返回用户登录信息
        用户名
        用户真实名称
        加密用户id(通过baes64加密  后续可以进行解密)
  前台
     js 请求后台登录接口
       1.参数校验
          用户名  密码 非空
       2.发送ajax 请求后台
       3.判断登录结果
           成功
             用户信息写入cookie
             执行跳转 (主页面)
           失败
             显示失败信息
2.主页面用户信息展示
    I.获取cookie 信息显示
       调用cookie.js 方法 操作dom
    II.后台查询  放入域对象
      解析用户cookie 信息  用户登录id 加密串解析
      根据用户id 查询用户记录 放入request
      执行转发
